# All Unity completes the non-USD FX marketplace with a regulated EUR leg — and shares the market maker already

> The stalled Agora/Noble/Plume 'non-USD FX marketplace' lacks a regulated EUR anchor; All Unity explicitly proposed EURAU-Stablebond FX pools and already runs on Kappa Markets — the same market maker Etherfuse uses — making it the missing, plumbing-compatible EUR leg of that same package.

The existing 'agora-noble-plume-are-one-stalled-non-usd-fx-marketplace' finding bundles three Nurture-stage leads describing one on-chain FX marketplace for non-USD stablecoins. All Unity independently asked for the same thing: on the January 2026 call, Manuel expressed interest in **'FX pools: EURAU-Stablebonds for on-chain FX market'** [wiki/entities/companies/all-unity.md]. Critically, All Unity's market maker is **Kappa Markets — the same MM Etherfuse uses for Mexican assets** [wiki/entities/companies/all-unity.md], so the liquidity plumbing for EURAU↔CETES/EUROB/TESOURO pools already shares an operator. The strategic note on the page even calls this out: 'The shared Kappa Markets relationship provides natural coordination' [wiki/entities/companies/all-unity.md].

What All Unity adds that Agora's AUSD (a USD stablecoin) cannot is a **BaFin-regulated, MiCAR-compliant EUR leg** backed by DWS/Flow Traders/Galaxy Digital [wiki/entities/companies/all-unity.md]. A non-USD FX marketplace whose only 'stable' anchor is a USD coin isn't really non-USD; adding a regulated EUR stablecoin (plus the HKD/AUD/CNY coins above) turns it into a genuine multi-currency FX venue where sovereign-yield Stablebonds are the settlement assets — exactly the 'on-chain FX' thesis Etherfuse markets [shared/master-knowledge-base.md]. All Unity is live on Base, Polygon, Arbitrum, Avalanche and expanding to Stellar and Solana [wiki/entities/companies/all-unity.md], covering the chains Etherfuse already deploys on.

Actionable synthesis: don't work All Unity's FX-pool interest as a separate thread — fold it into the same revival package as Agora/Noble/Plume, with All Unity as the regulated EUR anchor and Kappa Markets as the shared liquidity operator across all legs.
